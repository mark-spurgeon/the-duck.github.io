Install Duck Launcher

Install on Ubuntu based platforms

To install Duck Launcher on Ubuntu, you have to add the-duck's Launchpad ppa, then you can install it like any other program. To do that, simply type these commands on a terminal:

sudo add-apt-repository ppa:the-duck/launcher
sudo apt-get update
sudo apt-get install ducklauncher
