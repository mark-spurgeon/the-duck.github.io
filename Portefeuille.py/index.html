Intro / whatever goes here

Portefeuille.py is a static portfolio generator written in Python. Alright, I know there are plenty of static website, and portfolio, generators out there that probably do a better and more complete job. But there's no really simple solution out there. All I want is a downloadable script that just reads my portfolio folder and generates an HTML file. So I wrote this..

Simple ? / How's that..

You see, there's just one file, you download it, place it where you want your HTML file to be, place your folder with all your images in the same folder, then run python portefeuille.py. And that's it, you have your portfolio!

Well in fact / it's not as simple as that

As you might have noticed, Portfolio.py is focused on images and media. So basically it scans a folder containing images. These images, however, have to be stored in categories, sub-folders, covering something general like "mobile designs" or "videos". In these categories are collections, these are groups of images that cover a common project, like "Design for The Bakery Foundation".. Anything outside of a collection cannot yet be identified by Portfolio.py

This decision helps better organising a portfolio, and most portfolios do follow this order. Check out the demo if you don't believe me.

Let's get started / Shall we ?

First, download Python 2.7+, make sure you have pip installed, and install these modules (in a terminal): 

pip install PyYAML Jinja2

Finally, download portefeuille.py from github, copy this file to the folder where you want your HTML file generated. Add the folder containing your images, and you're ready to generate your portfolio!
