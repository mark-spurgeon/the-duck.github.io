Intro / whatever goes here

Portefeuille.py is a static portfolio generator written in Python. Alright, I know there are plenty of static website, and portfolio, generators out there that probably do a better and more complete job. But there's no really simple solution out there. All I want is a downloadable script that just reads my portfolio folder and generates an HTML file. So I wrote this..

Simple ? / How's that..

You see, there's just one file, you download it (and install dependencies, sorry!), place it where you want your html file to be, place your folder with all your images in the same folder, then run python portefeuille.py. If you want to add more information, your can add *.yaml files, follow the documentation

Well in fact / it's not as simple as that

So, you must follow a certain architecture. Let me explain, you have groups of images, collections, which are stored in categories, they must follow this architecture. This is how portfolios should be designed, in my opinion.
